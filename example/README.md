# Example Docusaurus Site

This is an example of how to use the Hoverkraft Docusaurus theme.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Configuration

The theme is configured in `docusaurus.config.js` with Hoverkraft branding colors and settings.
